Introduction

Portugol, also known as structured Portuguese, is a family of programming languages based on the Portuguese language. Some of its variations can be considered pseudocode, and others are complete, context-free languages with defined grammars and implementations in editors or compilers. They are used both to study algorithms and data structures and to create compilers, interpreters and diagramming tools such as flowchart generators.

History

There is conflicting information about the official creators of Portugol, both as a method and as an implementation. According to the blog archive of the Portugol dialect website VisuAlg, the effort was started as a translation of the Pascal programming language into Portuguese between 1980 and 1983 by Professor Antonio Carlos Nicolodi, Pascal being the direct descendant of the programming language called ALGOL, which was only used on mainframes. At the time, the dialect was only called ‘Portugol’ and had no implementation.

Portugol had its first implementation created in 1996 by Cláudio Morgado de Souza. This Pascal-based variation of Portugol is known as Portugol VisuAlg, implemented in its own editor. It is currently maintained by Nicolodi.

A second variation of Portugol was created by António Manso in 2004 and maintained until 2006, and is quite different from the VisuAlg variation, both in syntax and basic commands. For example, for an algorithm that writes ‘Hello world’ on the screen, we should write:


  algoritmo "ola-mundo"
    inicio
      escreva ("Olá mundo")
    fimalgoritmo

As for the variation of the Polytechnic Institute of Tomar (IPT), we write it as follows:


  inicio
    escrever "Olá mundo"
  fim

O nome Portugol é uma mistura das três palavras: Português, Algol e Pascal, (PORTUguês, ALGOL e PascaL).

Variations

Portugol has variations inspired by programming languages other than Pascal, such as C, C++ and Java. Portugol Studio, implemented by the technology centre at the Federal University of Vale do Itajaí (UNIVALI), and Mapler, developed by students at the Federal Institute of Maranhão, are Java-based implementations, also written entirely in Java. Other well-known variations are:

  • G-Portugol
  • Portugol Viana
  • P&G editor

Variations such as IPT and Mapler have functionalities in which the user can draw flowcharts and translate these diagrams into Portugol code. Mapler also has the ability to translate Portugol algorithms into high-level languages such as Java, Python and C++[6].

Portugol Viana is a variation of Portugol that continues the implementation of Portugol IPT, with commands, operators, functions, recursive functions, structures and constructors for complex structures such as stacks, queues, linked lists and doubly linked lists[7]. The author is from the Viana do Castelo School of Technology and Management.

G-Portugol is a Portugol compiler for executables. It has a compiler for Windows and one for Linux.

Portugol method

Portugol as a method or pseudo-language allows any Portuguese speaker to develop structured algorithms in a simpler and more intuitive way, regardless of the syntax of real programming languages. The method can be described as follows:

  • An algorithm is described as a series of steps, like, for example, a cake recipe;
  • The technique of successive refinements is employed, i.e. each step is refined to resemble an instruction that a machine can execute;
  • After the final refinement, the algorithm is coded in some context-free language.

For example: an algorithm to print out a series of numbers from 1 to 15 can be described below in natural language:

  • Define a variable with the value 1;
  • Write the variable;
  • Add 1 to the variable;
  • Repeat the last two steps until the variable has the value 15.

Transcribed into Portugol, we would have:


  inteiro variavel;
  variavel <- 1;
  enquanto variável <= 15
    escreva(variavel);
    variavel <- variavel + 1;
  fim enquanto;

It is easy to implement algorithms developed in Portugol by mapping them to the desired programming language. The method has been around since the 1970s and is used to learn algorithms and data structures. It is present in many programming teaching materials.

It is also used in conjunction with block diagrams (such as the Flowchart or Chapin Flowchart. Some variations of Portugol can generate code from flowcharts and vice versa.
